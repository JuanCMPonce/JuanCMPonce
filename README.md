Proyecto: Monitor Cardiaco
Lenguaje de Programacion Utilizado: C#

Este proyecto fue desarrollado en visual studio forms en el cual buscamos que mediante un sensor cardiaco configurado mediante un arduino y ensamblado en una protoboard los usuarios pudieran monitorear su ritmo cardiaco y verificar que su ritmo cardiaco fue el correcto, todo esto daria el resultado junto con la informacion fisica del usuario y con su complexion fisica obtendriamos el ritmo cardiaco del usuario y podriamos verificar si su ritmo cardiaco era el adecuado para su complexion o si estaba en riesgo de sufrir algun percanse cardiovascular
